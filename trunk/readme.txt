To build MbUnit installer.

- Minimum Requirements

1) .NET Framework 1.0 redist
2) .NET Framework 1.1 redist
3) .NET Framework 2.0 beta 2 redist

- Build MbUnit Installer

1) Launch 'Command Prompt'
2) execute "msbuild MbUnit.proj"
3) Look in build\ for MbUnit.msi
